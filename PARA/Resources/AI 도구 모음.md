## Claude Code 기본 명령어
- `/help` - 도움말 보기
- `/init` - CLAUDE.md 파일 생성
- `/config` - 설정 확인/변경
- `/clear` - 대화 초기화
- `Shift+Tab` - Plan 모드 전환

## 프롬프트 골든 공식 (RCIF)
1. **R**ole (역할) - AI에게 역할 부여
2. **C**ontext (맥락) - 배경 정보 제공
3. **I**nstruction (지시) - 구체적 작업 요청
4. **F**ormat (형식) - 출력 형식 지정
