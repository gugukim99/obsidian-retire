# Quick Review

## Description
Perform a quick code review on the current project or specified files.

## Instructions
When this skill is invoked:

1. Identify the files to review (changed files or specified files)
2. Read each file and analyze for:
   - Code quality issues
   - Potential bugs
   - Security concerns
   - Style inconsistencies
3. Provide a concise summary with actionable feedback

## Output Format
- List issues by severity (high / medium / low)
- Include file path and line number for each issue
- Suggest specific fixes where possible
